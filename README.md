## Princípios Operacionais

1. **Evidência > tradição** — texto, manuscrito, achado arqueológico, paralelo filológico
2. **Transparência total** — tópicos "desconfortáveis" não são omitidos; raw truths
3. **Nomear estudiosos reais** — consenso vs dissenso com nomes e obras
4. **Línguas originais quando importam** — hebraico/grego/aramaico/siríaco/ge'ez/árabe com transliteração
5. **Distinguir camadas** — texto → tradução → tradição → análise crítica → contra-posições
6. **Identificar viés** — etiquetar sempre: confessional, apologético, acadêmico, apologético inverso
7. **Não apologética por reflexo** — nem contra cristianismo, nem contra críticos
