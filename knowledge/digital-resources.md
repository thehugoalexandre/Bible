# Recursos Digitais — Software, Bibliotecas, Bases de Dados

Ferramentas para estudo bíblico acadêmico sem propaganda confessional obscurecendo a análise.

## Software Bíblico

### Logos / Verbum (Faithlife)
- **logos.com** / **verbum.com**
- **Padrão-ouro** para acadêmicos evangélicos e católicos
- Verbum = versão católica (inclui Magisterium, Thomas Aquinas, Catena Aurea)
- Biblioteca comercial vasta (texto crítico, léxicos BDAG/HALOT, comentários técnicos Anchor/Hermeneia/ICC, NA28/UBS5 com aparato, BHS/BHQ)
- Ferramentas: análise sintática, morfologia, fusão de versões, busca avançada, links entre obras
- **Caro**, mas oferece *Logos Free Edition* com funcionalidade limitada
- Acesso a Sefaria, CCEL e outras coleções mediante pacote

### Accordance
- **accordancebible.com**
- Historicamente Mac-first, agora cross-platform
- **Preferido por muitos hebraístas acadêmicos** (motor de busca forte)
- Módulos do **Westminster Hebrew Morphology**, Göttingen LXX, Stuttgart Vulgate
- Mais rápido e leve que Logos

### BibleWorks (**descontinuado em 2018**)
- Ainda tem comunidade ativa com arquivos legados
- Alternativa para quem já possui licença

### The Word (freeware)
- **theword.net** — **gratuito**, Windows
- Recursos gratuitos extensos (muitos módulos criados por comunidade)
- Excelente para estudar traduções em paralelo, léxicos antigos (Strong, Thayer, BDB)
- Referências patrísticas (ANF/NPNF)
- Pode baixar BHS, grego, LXX
- Menos polido que Logos mas **sem custo**

### e-Sword (freeware)
- **e-sword.net** — gratuito Windows/Mac
- Biblioteca modular extensa; muitas comentários livres
- Interface mais antiga mas funcional

### Olive Tree BibleStudy (mobile/desktop)
- **olivetree.com**
- Forte em mobile; integração Logos

### STEP Bible (Tyndale House)
- **stepbible.org** — **gratuito, web-based, código aberto**
- Desenvolvido pela Tyndale House Cambridge
- Texto: THGNT (Tyndale House Greek NT), NA28, BHS, versões
- Interlinear, léxicos, acessível sem instalação
- **Excelente para começar sem investimento**

### MySword / AndBible (Android)
- Apps gratuitos para Android

### Sefaria (judaico)
- **sefaria.org** — **gratuito, código aberto**
- **Base de dados judaica** mais completa online
- Tanakh com todos os comentários rabínicos (Rashi, Ibn Ezra, Ramban, etc.)
- Mishná, Talmud Bavli completo (texto hebraico + tradução Steinsaltz inglesa), Yerushalmi, Midrash Rabbah, Tosefta, Maimonides, Shulchan Aruch
- Kabbalah: Sefer Yetzirah, Zohar (em progresso)
- APIs para desenvolvedores
- **Indispensável** para estudo rabínico

## Advertências

1. **Nem tudo "gratuito" é acadêmico** — muitos sites amadores propagam teorias conspiratórias, apologética confessional disfarçada, leituras fundamentalistas ou esotéricas
2. **Filtrar por afiliação institucional** — PhDs de universidades seculares ou católicas pós-V2 são mais confiáveis para histórico-crítico que sites apologéticos
3. **Verificar múltiplas fontes** — especialmente em tópicos controversos (datação, autoria, historicidade)
4. **Wikipedia** — útil como ponto de entrada; verificar referências; páginas biblícas variam em qualidade (algumas excelentes com editores acadêmicos ativos, outras ruins)
5. **Para Islã**, particularmente, separar rigorosamente apologia (cristã ou muçulmana) de academia (IQSA, Brill Encyclopedia of the Qur'an)

## Combinação Recomendada (sem custo)

1. **STEP Bible** — para texto crítico + léxicos
2. **Sefaria** — para tudo judaico
3. **earlychristianwritings.com** — para contexto primitivo
4. **CCEL** — para patrística
5. **tertullian.org** — para textos obscuros
6. **Archive.org** — para clássicos do séc. XIX/XX
7. **Bible Odyssey** + **TheTorah.com** — artigos acadêmicos público geral
8. **Academia.edu** — para papers de especialistas
9. **JSTOR** (via biblioteca pública ou JPASS) — para periódicos

### Com investimento
1. **Logos** ou **Accordance** — biblioteca acadêmica consolidada
2. **Brill reference works** (se institucional)
3. **Hermeneia, Anchor Yale, ICC** — comentários padrão por livro
